# Plataforma de Gestão de Tarefas empresariais com Chat Integrado

> Status: Em desenvolvimento;

## Objetivo
Criar uma plataforma centralizada para otimizar a distribuição de demandas corporativas e a comunicação interna em tempo real.

## Stack Tecnológica (Planejada/Em Uso)
* **Backend:** Node.js, TypeScript, Express, Prisma ORM
* **libraries auxiliares do BackEnd:**  (bcrypt, cookie, cookie-parser, cors, jsonwebtoken, dotenv, http, nodemon, zod)
* **Comunicação em Tempo Real:** Socket.io, ws(WebSockets)
* **Frontend:** React, Bootstrap, socket.io-client, axios
* **Infraestrutura:** Docker, PostgreSQL
